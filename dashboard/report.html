  • Core Engine Correctness Crisis: Base scheduler produces non-deterministic outputs at temperature=0 (#39146, 10/10 reproduction), V1 scheduler deadlocks under sustained load (#37729, 15 comments), and prefix cache corruption affects multiple model families.   • Tool Calling & Structured Output: Responses API streaming objective blocked by tool parser regression (#39056, 11 comments), MTP generating malformed JSON (#34449), and structured output constraints bypassed when reasoning enabled (#34650). Production AI agent workflows broken since v0.18.
